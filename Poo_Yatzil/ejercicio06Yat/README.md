# 🧟 Sistema de Mobs en Python

Este proyecto simula distintos mobs inspirados en Minecraft, mostrando su sonido, comportamiento y forma de moverse.

## 🚀 Funcionalidad

* Crear mobs (`Vaca`, `Creeper`, `Enderman`, `Aldeano`) con nombre y nivel
* Presentar cada mob mostrando:

  * Sonido característico
  * Comportamiento
  * Forma de moverse

## ▶️ Ejecución

```bash id="mobs01"
python main.py
```

## 📌 Ejemplo de uso

Se crean y presentan cuatro mobs:

* Vaca
* Creeper
* Enderman
* Aldeano

Cada mob se presenta mostrando información relevante en consola.

## 🛠️ Requisitos

* Python 3
