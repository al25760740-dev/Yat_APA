# ⛏️ Sistema de Herramientas de Minecraft en Python

Este proyecto simula un sistema de herramientas inspiradas en Minecraft. Cada herramienta tiene un tipo, nivel de durabilidad y puede usarse sobre diferentes objetivos hasta romperse.

## 🚀 Funcionalidad

* Crear herramientas (`Pico`, `Espada`, `Pala`) con material y durabilidad
* Usar herramientas sobre objetivos específicos (`mena de diamante`, `Creeper`, `Arena`)
* Mostrar estado de la herramienta después de cada uso
* Detectar cuando una herramienta se rompe

## ▶️ Ejecución

```bash id="tools01"
python main.py
```

## 📌 Ejemplo de uso

Se crean tres herramientas:

* Pico de Diamante
* Espada de Hierro
* Pala de Madera

Cada herramienta se usa repetidamente sobre un objetivo hasta que se rompe, mostrando mensajes de uso y estado.

## 🛠️ Requisitos

* Python 3
