# ⚠️ Manejo de Excepciones en Python

Este programa demuestra el uso básico de manejo de errores en Python utilizando `try`, `except`, `else` y `finally`.

## 🚀 Funcionalidad

* Solicita dos números al usuario
* Realiza una división
* Maneja errores comunes como:

  * Entrada inválida (`ValueError`)
  * División por cero (`ZeroDivisionError`)
* Muestra el resultado si no hay errores
* Ejecuta un mensaje final siempre

## ▶️ Ejecución

```bash id="exc01"
python main.py
```

## 📌 Ejemplo

El usuario ingresa un numerador y denominador.
El programa valida los datos y evita que falle si ocurre un error.

* Python 3
