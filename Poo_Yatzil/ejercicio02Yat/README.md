# 🏦 Sistema de Cuenta Bancaria en Python

Este programa implementa una clase `CuentaBancaria` que permite manejar depósitos, retiros y consultar saldo de un usuario.

## 🚀 Funcionalidad

* Crear cuentas bancarias con titular, número de cuenta y saldo inicial
* Depositar dinero y actualizar el saldo
* Retirar dinero con validación de saldo
* Consultar saldo actual
* Mostrar información del titular y su saldo

## ▶️ Ejecución

```bash id="bank01"
python main.py
```

## 📌 Ejemplo de uso

```text
Yatzil Aparicio:  tienes 1000.0
Deposito de 500
Nuevo saldo: 1500.0
Intento de retirar 1500
Resultado: 1500
Saldo actual: 0
```

## 🛠️ Requisitos

* Python 3
