# 🐾 Sistema de Mascotas en Python

Este programa implementa una clase `Mascota` que permite crear mascotas virtuales y simular interacciones como alimentarlas y jugar con ellas, controlando su nivel de felicidad.

## 🚀 Funcionalidad

* Crear mascotas con nombre, tipo, edad y nivel de felicidad
* Validar que la felicidad esté entre 0 y 100
* Alimentar y jugar para aumentar la felicidad
* Consultar el estado actual y saber si la mascota es feliz

## ▶️ Ejecución

```bash id="pet01"
python main.py
```

## 📌 Ejemplo de uso

Se crean dos mascotas:

* Doky (Perro, felicidad 50)
* Tomy (Gato, felicidad 80)

Luego se interactúa con ellas, mostrando su estado y felicidad después de alimentar o jugar.

## 🛠️ Requisitos

* Python 3
